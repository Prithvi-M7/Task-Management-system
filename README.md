 Task Management System (Java)

 Project Overview
The Task Management System is a simple Java console-based application that allows users to manage tasks efficiently.  
Each task has an ID, title, description, and priority.

This project demonstrates core Java fundamentals, Object-Oriented Programming (OOP), and basic data structures.

 Features
- Add new tasks
- Store tasks using Java Collections
- Display tasks sorted by priority
- Uses multiple Java classes

 Technologies Used
- Language: Java
- Concepts:
  - Object-Oriented Programming
  - Encapsulation
  - Java Collections (ArrayList)
  - Sorting using Comparator
- Tools:
  - VS Code
  - Java JDK
  - Command Line / PowerShell

Project Structure
TaskManagementSystem/
- Main.java          (Entry point)
- Task.java          (Task model)
- TaskManager.java   (Task operations)

 How to Compile and Run

1. Open terminal in the project folder
2. Compile:
   javac Main.java Task.java TaskManager.java
3. Run:
   java Main
